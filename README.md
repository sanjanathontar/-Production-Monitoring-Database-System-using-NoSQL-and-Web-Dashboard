# -Production-Monitoring-Database-System-using-NoSQL-and-Web-Dashboard

The Production Monitoring Database System is a smart manufacturing analytics solution developed to monitor production activities, machine downtime, and shift performance using a NoSQL database and an interactive web dashboard. The system enables efficient storage, retrieval, and analysis of production data, helping industries improve operational efficiency and make data-driven decisions.

The project utilizes MongoDB as the NoSQL database to manage production records, downtime information, operator details, and shift-wise performance data. Analytical queries are implemented to calculate production efficiency, identify downtime causes, and evaluate productivity trends. A user-friendly dashboard provides real-time visualization through charts, reports, and performance indicators.

Key Features

Production count monitoring and tracking

Machine downtime recording and analysis

Shift-wise performance management

Daily efficiency calculation and reporting

Interactive analytics dashboard

NoSQL database integration using MongoDB

Data visualization with charts and graphs

Scalable and flexible database architecture
